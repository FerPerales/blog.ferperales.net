--- layout: post title: "\n\t\t\t\tGithub y sus 3 millones de usuarios\t\t" --- Hace casi una semana se anunciaba con gran emoción en el blog de Github que se habían alcanzado 3 millones de usuarios registrados en la plataforma logrando un importante crecimiento a lo largo de sus casi 5 años de existencia. ¿Qué es Github? Github es servicio web de hospedaje de proyectos de software basado en el sistema de control de versiones Git. Considero que gran parte de la popularidad del servicio es debido a sus toques de "red social" para programadores permitiendo seguir a nuestros amigos o proyectos que son de nuestro interés así como contribuir a los mismos reportando bugs, realizando mejoras al código fuente o tomar el proyecto como base para nuestros intereses ya que los repositorios públicos son liberados como proyectos open source. Otro aspecto resaltable de Github es que, de alguna forma, se ha convertido en una especie de portafolio para programadores (o al menos una carta de presentación sobre tu trabajo) ya que cada vez es más común encontrar ofertas laborales donde te piden enviar un enlace a tu cuenta de github para ver algunos de los proyectos en los que has trabajado con anterioridad convirtiendo con esto a github en algo así como lo que es deviantart para los diseñadores o flickr para los fótografos: una plataforma para dar a conocer lo que haces. Tengo aproximadamente un año de usar Github a raíz de que lo usamos para Sharealla en Startup Weekend Guadalajara y, desde ese entonces, he dedicado algo de tiempo libre para aprender a usarlo para subir algunos proyectos escolares y otros que hago en mi tiempo libre. A pesar de que la calidad de mi código puede mejorar bastante, decidí subir los proyectos tal cual y mejorarlos en los ratos libres. Otra de las razones por la que deberías pasar algún tiempo en Github es por la enorme cantidad de buenos proyectos que hospeda y de los cuales se puede aprender mucho al estudiar su código: el kernel de linux, el framework Ruby on Rails, el framework para front-end bootstrap, el gestor de paquetes para OS X homebrew, node.js, jQuery e incluso el código fuente de la red social diaspora esta disponible. Desconozco si Github ya se ha posicionado como la mayor fuente de open source en internet pero, al ritmo que van, no me extrañarían que en este año logren llegan a los 6 millones de repositorios y 4 millones de usuarios registrados así que, ¿qué esperas para formar parte de Github? Por último un pequeño detalle anecdótico: hace algunos meses me pidieron desarrollar una plataforma de crowfunding pero tuve que rechazar la propuesta por falta de tiempo. Un mes después de eso me encontré con selfstarter.us: un proyecto open source creado en Ruby on Rails para montar tu propia plataforma de crowfunding que sin duda me hubiera ayudado mucho conocer.
